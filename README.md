Aplicação de web scraping (raspagem de dados) em uma página da Amazon para o produto "iPhone". É possível alterar para qualquer outro produto. É retornado um arquivo .csv com a listagem do produto procurado.

* Python
* Pandas
* Beautiful Soup
* CSV
